# Visitor Counter App

## Overview

This application counts the number of unique visitors and displays this statistic. The backend is developed with Flask (Python) and uses PostgreSQL as the database.

## Technologies Used

- Flask (Python)
- PostgreSQL
- Docker
- Docker Compose
- Terraform
- Azure DevOps

## Setup Instructions

### Prerequisites
